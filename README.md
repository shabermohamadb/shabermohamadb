> *I don't just write code. I architect experiences — from the logic inside a game engine to the circuits on a breadboard, from a screenplay to a training dataset. If it can be built, it can be reimagined.*

I'm **Shaber Mohamad B** — a multi-stack developer studying **Artificial Intelligence & Data Science** at **Stella Mary's College of Engineering**, Tamil Nadu. My work sits at the intersection of four high-voltage disciplines:

- 🎮 **Game Tech** — Designing interactive mechanics, immersive systems, and player experiences using Unity & C#. I don't build games; I engineer worlds.

## 🏗️ ARCHITECTURAL PROJECTS

### 🕹️ Laser Shooter — Physical Game Interface
> *Arduino Uno × LDR Sensors × Custom Game Logic*

A fully custom game crossover where physical hardware **becomes the controller**. LDR sensors detect laser hits in real-time, feeding live data into a C#/Unity pipeline. The result: a tangible, room-scale shooting game that blurs the line between arcade hardware and software simulation.

- **Tech**: Arduino Uno, LDR sensor array, serial communication bridge, Unity game loop
- **Impact**: Demonstrates hardware-software state synchronization at sub-100ms latency

---

### 🏡 Smart Ecosystem — IoT Home Automation
> *ESP32 / ESP8266 × MQTT × Sensor Integration*

A connected home architecture built from scratch — controlling appliances, reading ambient data, and responding to environmental triggers over Wi-Fi. Designed for modularity: swap sensors, add nodes, scale rooms.

- **Tech**: ESP32 / ESP8266, DHT sensors, relay modules, MQTT protocol, Wi-Fi OTA
- **Impact**: End-to-end smart home prototype with real-time remote control capability

---

### 🎬 Viva 404 — Digital Short Film Project
> *Screenwriting × Visual Direction × Asset Design*

A creative directorial project exploring the human experience through a digital-native lens. **Viva 404** is more than a short film — it's a production pipeline: original screenplay, custom visual assets, sound design intent, and character direction.

- **Tech**: Screenplay structure, Figma (storyboarding), Photoshop (visual assets), narrative systems design
- **Impact**: Proof that a developer's mindset elevates storytelling — every scene is a system, every arc a logic tree
